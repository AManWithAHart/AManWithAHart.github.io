Cloth Simulation

Coded in Personal or Class Year Completed
Processing Class: CSCI5611 Animation and Planning in Games 2023

Using position based dynamics and processing's graphics engine, I worked with a classmate to create a cloth that would follow basic physics and would have object collision. In addition, we added a toggle to add texture to the cloth.

This project was the first project I had encountered that had me applying more dynamic physics to a coding project. I learned how using basic physics, we can take minor shortcuts in order to simulate the same property.

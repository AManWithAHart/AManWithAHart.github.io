Animation and Hierarchical Transformations

Coded in Personal or Class Year Completed
TypeScript Class: CSCI4611 Programming Interactive Computer Graphics and Games 2023

Using a series of interconnected matrices, I created a skeleton that was used to wrap objects around. I then read in mocap data to apply movements to the skeleton to create movements.

This project taught me how to use transformations to scale, rotate and translate shapes into characters that we can apply movements to. In addition, I also learned how to use matrices in a hierarchy to, in this case, create an animated character.
